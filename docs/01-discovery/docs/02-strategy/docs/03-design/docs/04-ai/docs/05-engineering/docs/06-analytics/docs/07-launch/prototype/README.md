# Prototype

## Overview

This folder contains the working prototype of the AI-Powered Intent Search for YouTube.

The prototype demonstrates how Large Language Models (LLMs) can improve YouTube search by understanding user intent rather than relying only on keyword matching.

---

## Objectives

- Demonstrate AI-powered search
- Understand user intent
- Retrieve relevant YouTube videos
- Re-rank search results using AI
- Explain recommendations
- Enable conversational follow-up queries

---

## Planned Features

- Intent classification
